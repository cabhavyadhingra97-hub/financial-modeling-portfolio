Full 5-year 3-statement forecast with segment analysis (O2C, Retail, Jio Digital, Oil & Gas).

**Skills:** DCF · FCFE · WACC · Residual Income · Segment Analysis · 3-Statement Model

### 3. HDFC Bank — Equity Research Report
> Published a buy-side equity research report using the CAMEL framework. Includes shareholding pattern analysis and technical analysis with support at Rs. 1,590 and resistance at Rs. 1,701.

**Skills:** Equity Research · CAMEL Analysis · Peer Benchmarking · Technical Analysis · Banking Sector

### 4. LBO Model — Leveraged Buyout Analysis
> Structured a full LBO model with a 4-year hold (2021–2025): 12x Net Debt/EBITDA leverage, 10% annual revenue growth, 40% EBITDA margin, 25% sponsor return hurdle rate. Key outputs: Exit EV = 89,310, Exit Equity Value = 40,811, Max Sponsor Investment = 16,716. Debt schedule with required + optional paydown modelled annually.

**Skills:** LBO · Debt Schedule · Entry/Exit Multiples · Sponsor IRR · Financial Sponsor Analysis

### 5. Accretion / Dilution — M&A Analysis

New debt of $6.3B at 5%, $100M synergies, $500K incremental D&A, $5M advisory fees. Result: Pro Forma EPS = $25.90 vs. standalone $25.00 — deal is 3.61% accretive.

**Skills:** M&A Modelling · Accretion/Dilution · Pro Forma EPS · Deal Structuring · Synergy Modelling
