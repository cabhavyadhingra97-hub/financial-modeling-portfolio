# Project Finance Model — Edible Oil Refinery

## Project Overview

**Type:** Greenfield Project Finance Model
**Sector:** Agri-Processing / Edible Oil Manufacturing
**Project Cost:** Rs. 198+ Crore (Total Project Outlay)
**Revenue at Peak (FY31):** Rs. 2,320 Crore
**Model Complexity:** 15+ Interconnected Excel Sheets

---

## Business Problem

A promoter group planning to set up a large-scale edible oil refinery in India needed a comprehensive bankable project report and financial feasibility model to:
- Evaluate viability across 3 product lines simultaneously
- Structure an optimal debt-equity mix for term loan sanction
- Demonstrate debt serviceability (DSCR) to lenders
- Compute project-level IRR for equity investor decision-making

---

## What I Built

A full integrated Project Finance model in Excel with 15+ interconnected sheets covering every aspect of a real-world bankable DPR (Detailed Project Report):

### Model Architecture

| Sheet | Purpose |
|---|---|
| **Inputs** | Centralised assumption dashboard (capacity, prices, interest rates, D/E ratio) |
| **Site Development** | Land & civil cost breakdown (Rs. 800 Lakhs) |
| **Plant & Machinery** | Equipment cost schedule (Rs. 149 Crore, 34 line items) |
| **Capacity** | Production computation across 3 oil types |
| **Production** | Year-wise output with capacity ramp-up (50% to 95%) |
| **Revenue** | Sales projections with 2% annual price escalation |
| **Raw Material Cost** | Material cost model with 6 input types and yield ratios |
| **Power & Fuel** | Utility cost model (4,680,000 KWH at full capacity) |
| **Employee Costs** | Payroll model for 82 employees with 5% annual increment |
| **Repair & Maintenance** | Asset-linked maintenance cost computation |
| **Fixed Assets (FAS)** | Depreciation schedule — 3 asset classes, WDV method |
| **IDC** | Interest During Construction — quarterly drawdown schedule |
| **Term Loan Schedule** | Monthly amortization, 12% p.a., full 11-year schedule |
| **Working Capital** | Bank finance requirement with margin money computation |
| **P&L Statement** | 9-year projected Income Statement |
| **Balance Sheet** | 9-year projected Balance Sheet |
| **Cash Flow Statement** | 9-year projected Cash Flow Statement |
| **DSCR** | Debt Service Coverage Ratio — year-wise and average |
| **IRR** | Project IRR and Return on Capital Employed |
